Planejamento

12/05 - 31/05 - Finalizar pesquisa preparatória

Ao fim desse período, obter uma compreensão completa das arquiteturas utilizadas nos papers DanceDanceConvolution e DanceDanceConvLSTM, incluindo as arquiteturas LSTM, Peephole LSTM, ConvLSTM, Redes Neurais Profundas, Redes Convolucionais, e extração de features de áudio. Além disso, compreender apropriadamente a arquitetura transformer utilizada no paper Yi2023. Finalmente, compreender a arquitetura conformer para aplicar futuramente.

01/06 - 14/06 - Aplicar arquiteturas de DDC e DDCL para Pump it Up, Início de Escrita de Monografia

Começar a escrita da monografia, trazendo uma introdução sobre os objetivos do projeto, trabalhos prévios (DDC, DDCL, Yi2023) e conceitos básicos de processamento de áudio utilizado.

Neste período, o objetivo é adaptar a pipeline de extração de músicas do DDCL para Pump it Up, e treinar modelos para geração de Pump it Up utilizando as arquiteturas LSTM apresentadas nos papers DDC, DDCL.
